romance set in a modern college AU of Genshin Impact. The player is **Aether**, a

## 🌱 Conventions (read before adding docs)

The cast is large and intentionally **lazy-loaded** — characters are introduced only
when Aether's path crosses theirs. To keep this bible aligned with that rule:

1. **Full character docs exist only for characters who have appeared** in the story.
   Everyone else lives as a roster entry in [`characters/_index.md`](characters/_index.md)
   marked *Not yet appeared*.
2. **When a new character appears**, create `characters/<name>.md` from the template
   used by existing docs, and flip their status in the roster index.
3. **When a new place is established**, add `locations/<name>.md` and list it in the
   location index.
4. **After each chapter**, save the prose under `chapters/` and update affected docs
   (relationship status, affection, running threads, appearance logs).
